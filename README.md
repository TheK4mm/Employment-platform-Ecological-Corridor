# Employment-platform-Ecological-Corridor

Este repositorio contiene un avance inicial (front-end mock) de una aplicación web para gestionar empleo en el Corredor Ecológico de Villavicencio.

## ¿Qué contiene este avance?
- `index.html`: página principal con registro, lista de ofertas y formulario para publicar (simulación).
- `styles.css`: estilos base.
- `script.js`: lógica front-end simple que guarda datos en `localStorage` como mock.
- `schema.sql`: esquema inicial en MySQL para la versión backend.
- `informe_apa_borrador.md`: borrador de estructura del informe final.
- `manual_usuario_borrador.md`: borrador del manual de usuario.
